Easily display notices on WooCommerce category pages you specify. Give your customers information on sales, stock issues, news, and more.

== Description ==

Giving your customers relevant information when you have sales, inventory problems, or important information related to the category your customers are browsing is made easy with Product Categories for WooCommerce.

Free Version:

* Target multiple categories
* Allows a, em, strong, and span html tags

[Enhanced Version](https://www.polyplugins.com/product/category-enhancements-for-woocommerce/ "Category Enhancements for WooCommerce Pro by Poly Plugins"):

* Add multiple notices to specified categories
* Add notices to products in specific categories
* Prevent checkout in specified categories based on state
* Modify category prices
* Wysiwig editor (Coming Soon)

== Installation ==


1. Backup WordPress
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Navigate to Products->Category Notices to configure the plugin

== Frequently Asked Questions ==

= How do I add a notice? =

Simply navigate to Products->Category Notices via your backend and add your categories and your notice.

= How do I remove a notice? =

Simply navigate to Products->Category Notices and delete the content of the fields or you can simply deactivate the plugin.
